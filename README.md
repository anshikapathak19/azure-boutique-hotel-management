# AzureStay 🏨

**Luxury Redefined. Seamlessly Connected.**

AzureStay is a premium cloud-native Boutique Hotel Booking and Management System built using React, Node.js, Express, and Microsoft Azure.

## ✨ Features

- Guest Authentication
- Luxury Room Booking
- Personalized Customer Preferences
- Admin Dashboard
- Room Management
- Booking Management
- Cloud Storage for Room Images
- Real-time Booking Updates
- Azure Cloud Integration

## ☁️ Microsoft Azure Services

- Azure Cosmos DB
- Azure Blob Storage
- Azure Functions
- Azure App Service / Azure Virtual Machine

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Lucide React

### Backend
- Node.js
- Express.js
- JWT Authentication

### Cloud
- Azure Cosmos DB
- Azure Blob Storage
- Azure Functions
